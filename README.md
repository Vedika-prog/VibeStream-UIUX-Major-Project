VibeStream is a mood-driven, season-inspired music streaming application designed as a Major UI/UX academic project. The goal of this project was to create a visually calming yet emotionally dynamic music experience that adapts to both user mood and seasonal themes.

The application was designed entirely in Figma using a component-based design system, structured layouts, and interactive prototyping. The primary focus of this project was to demonstrate strong understanding of UI principles, reusable components, visual hierarchy, auto layout, interactive transitions, and scalable design systems.

Rather than designing a static interface, this project explores how visual themes and emotional states can influence digital music experiences.
